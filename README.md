# Stardance API
An API for Hackatime's Stardance event.

Main API link: https://stardance-api.ultimatecraw.xyz/docs#/


## Features
- Regular 6 hour updates
- Easy-to-use endpoints
- Accurate project rankings
- Queryable data
- Historic data collection

## How it works
FastAPI works in Python to scrape project data direct from stardance's website.
